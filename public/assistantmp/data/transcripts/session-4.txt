Демо: работа с телефона через Telegram-бота:
Дан показал свою систему — Telegram-бот, подключённый к бесконечной сессии Claude Code на VPS ($25/мес, Digital Ocean, Сан-Франциско). Сессия не умирает благодаря скрипту, предотвращающему уход в спящий режим. Через бота можно делать всё, что можно делать на компьютере: управлять задачами в Linear, создавать события в календаре, работать с n8n workflow, разрабатывать и тестировать ботов, документировать в Obsidian. Дан перестал вручную создавать задачи в task-tracker и события в календаре — всё через бота.

Уровни памяти:
1. Встроенная память Claude Code — автоматическая, но привязывает к инструменту (информация зашифрована, нет контроля).
2. Obsidian-документация — файлы на компьютере, полный контроль. Структура: видение на 6 лет → стратегия на 6 месяцев → недельные файлы → daily-заметки. Дан ведёт заметки с 2019 года.

Obsidian:
Obsidian — это просто редактор markdown-файлов. Открывает папку с .md файлами и представляет в удобном виде. Не хранит данные в своём формате. К Apple Notes нет API, поэтому Obsidian — лучший выбор для работы с агентом. Capturing с телефона — через Telegram-бота командой «send» + голосовое сообщение → транскрибируется → сохраняется в daily-заметку.

n8n:
Платформа для автоматизации и разработки чат-ботов (low-code/no-code). У n8n есть API, через который Claude Code может: просматривать workflow, смотреть историю запусков, находить ошибки, разрабатывать и дорабатывать сценарии. Тестирование тоже через агента — второй Telegram-аккаунт, через который агент тестирует ботов.
