Объяснение концепций:
- API (Application Programming Interface) — программный интерфейс, позволяющий делать то же самое что через UI, но без визуального интерфейса, через запросы
- JSON — структура данных с фигурными скобками, внутри пары ключ-значение (client_id, client_secret и т.д.)
- MCP (Model Context Protocol) — протокол взаимодействия AI-агентов с внешними сервисами
